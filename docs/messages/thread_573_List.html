From: rscarpen (letterboxing@atlasquest.com) | Date: 2005-03-04 18:49:31 UTC

> I do the same thing on AQ on My Page. Just to see if any have been
> newly listed that we can get a Find credit for.

AQ does have an option where you can be notified of new boxes in your
area. Do a search like you normally would. If you don't specify a
city but do include the state and country, it'll list all boxes in a
given state.

Then, in the results page, there's a link in the top-right corner that
allows you to save the search as a "Favorite Search". With that,
there's an option to be notified when new boxes are listed that show
up in your favorite search.

That way, you don't have to troll the What's New? list to see if
anything new is of interest. AQ will send you mail when something you
might be interested in is listed.
