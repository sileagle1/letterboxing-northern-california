From: Gwen and Don Jackson (foxsecurity@earthlink.net) | Date: 2005-09-03 07:34:41 UTC-07:00
Kel
Dogs are not prohibited, but it would be extremely hot for them. If you take your dog make sure you water it frequently and wet it's ears and feet. We have never seen any rattlesnakes, but I always presume that snakes are about.
There are tons of boxes along the way, but if you would like a D&G box with only one visitor in three years then you should look up Dumbledore's Western Adventure. I think the code key throws folks off, but you probably already know the key or atleast where to find it. Have a great trip!
Don
