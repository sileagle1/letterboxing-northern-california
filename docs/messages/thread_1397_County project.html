County project

From: Lea Shangraw Fox (princesslea@alamedanet.net) | Date: 2006-11-01 01:36:05 UTC
As a result of Spring Chick's new contest, I decided to conduct an audit of how many
California counties have letterboxes. After some analysis, I believe that there are 4
counties lacking boxes. However, there might be some AQ only boxes (where county info
is not provided) and/or mystery boxes in some of the counties. Anyway, here's the list:

Colusa County
Lake County
Sierra County
Yuba County (curiously, Yuba City which has 8 boxes is not in Yuba County)

Also, Kings County has one box listed; it was by Tara, but clues are no longer available.
Don't know the status of the actual box.

And finally, the following counties have only 1 box, as far as I can tell:

Amador County
Glenn County
Imperial County
Lassen County

So I think the point of Spring Chick's exercise is to fill in all the counties. You guys up for
it?

-- Princess Lea
