> HI everyone I just wanted to know how I go about posting a web page
> or whatever it is if I plan on planting a box? Is there somewhere
> special I need to go to get started?

Posting a webpage is pretty ambitious--you probably just want a place
to distribute clues--which are probably the same places you *find*
clues. Both letterboxing.org and atlasquest.com allow you to post
clues. To start on letterboxing.org, from the main page, click on the
link to "Submit Clues" and follow the directions. On Atlas Quest,
there's a link to "Add Planted Box" under the Letterboxes menu item.
Most of the fields are self-evident, but there's are Help links in the
upper-right corner of the all the pages if you need more information.

Happy trails!

-- Ryan
