Hi all,

We went to look for the Schwinn Bike box yesterday. We found the
hiding place (under a big pile of rocks) but it appears to be a nesting
spot for some baby rattlesnakes. Fortunately we looked before reaching
in. The clues did mention looking out for rattlesnakes but we were not
expecting them to be quite so close! We did wait see if any of them
were going to leave but they did not, so we decided to skip this one at
least for now. I guess they will probably move on at some point, but
use extra caution (or a really long stick!) if you want to check it out
for yourself.
