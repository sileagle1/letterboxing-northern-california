From: Hedglin, Nils A (Nils.A.Hedglin@Intel.Com) | Date: 2005-08-04 08:38:54 UTC-07:00
Here's a bullet from the LbNA FAQ.

"A box can be counted as Found more than once if the preceding apply,
and its clue, location, and stamp image have all changed since Found the
first time."

So according to this, the Judah box wouldn't be a counted as a Find for
those that found it at the GRR (DoubleSaj & Old Blue were always the
rebels). But, this begs the question, if the stamp, location & clue has
changed, what's left to make it the same box? I guess the log book &
physical box could be the same, but if it's a new stamp, I'd assume
there'd be a new log book. So, I don't see how all these things could
change & still be somehow considered the same box. I guess if a box had
been vandalized & the stamp was lost, but the book was still there, then
the box could be moved to a better hiding spot & a new stamp carved.
The "new" box would still have some continuity with the original box,
but count as a new find.

I generally count something as a new Find if it is a new stamp in a new
location, even if the clues are still sorta the same. For example,
Amanda's original Dirty Harry box went missing. When she recarved it,
she used a different image and put it in a slightly safer location, so
I counted it as a new find. This Judah one is trickier though - same
stamp, new location, new clues. Since I didn't really do the work the
first time I found it (stumbled across it while I was looking for a
totally different box in Sacto) maybe I shouldn't have counted it the
first time since I didn't do the work of solving that long and
complicated puzzle! As you all know, we do what we want when we are
deciding what to count, so it's up to you to decide what is "right."
