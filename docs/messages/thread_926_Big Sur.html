Amanda from Seattle has a box at Jade Cove, I believe, if you get down
there.

I found several small pieces of nephrite jade at Jade Cove when I was
there. It's among serpentine, so there's lots of confusing green rock
there. The trick is to wear polarized sunglasses, which help make the
jade stand out.

I always say that Big Sur is 'painfully beautiful'. So gorgeous that
it hurts.

Enjoy!

Paul in SF
