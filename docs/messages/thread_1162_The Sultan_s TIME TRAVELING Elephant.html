Yes I have heard of this time traveling Sultan's Elephant, but had
never actually see it until you so kindly post the web page. Time
machines come in many forms, so this enormous mechanical elephant does
not surprize me. Jules Verne's original California time machine looked
like a wheel-less carriage with a 1950's Cessna 2-seat cockpit. It was
not particularly portable, but it did have small retractable wheels so
it could be moved manually short distances. Jules gave me this machine
many years ago and I had it converted into an AC Cobra automobile.

And no, it does not need to be going 88 mph in order to time travel.

Salute Jules Verne!...he was my friend.
