Hmm... I've never tried a virtual box before and haven't played
around with this one, but you are right that many website URLs are
case sensitive. (Atlas Quest, for instance.)

Most URLs try to use all lowercase--the defacto standard. If that
doesn't work, try capitalizing the first letter of each word. For
instance, instead of "messagenum" try "messageNum". Or if it's the
name of a directory, try "MyStuff" instead of "mystuff". And variations.

I don't know the URL in question you're working with so that may or
may not be the problem. If your browser history isn't letting you
change the case sensitivity, you should be able to delete the
browser's history completely from "Options" under the "Tools" menu
item of your browser menu.

If all else fails, ask the owner of the website or the person who made
the virtual box what's going on--whichever is most appropriate. ;o)
