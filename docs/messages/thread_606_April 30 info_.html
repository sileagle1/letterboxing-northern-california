RE: [LbNCA] April 30 info?

From: Hedglin, Nils A (Nils.A.Hedglin@Intel.Com) | Date: 2005-03-17 13:24:52 UTC-08:00
The event is being held in Old Sacramento. Currently there is only 1
box in Old Sac. The plan is that a number of people will place boxes in
Old Sac within a few days prior to the event. It's about 5 blocks long
& 3 blocks wide, so everything will be walkable. When they did a
gathering like this at Golden Gate Park, things were so spread out, I
think people only got a few boxes, so they wanted to keep this one
small. Some of the boxes will be there only for that 1 day, some will
be left there after the event is over, some will be transplanted to a
new site afterwards. There will also be an event stamp, a special stamp
carved just for this event that will not be available anywhere else.
