Black Diamond Mines Report

From: grumpygrinchy (ffuselier@comcast.net) | Date: 2006-04-24 23:12:42 UTC
We decided today was a perfect day for a hike so we headed to the
Black Diamond Mines Regional Preserve. We checked on Charming
Chapeau and found it dry and well, though a bit lonely. No one has
visited since last October.

Sedate Sedan is missing. It will not be replaced.

Found the correct tree for the Black Diamond Letterbox but found no
box. There has been a lot of limb pruning done recently on this
tree. We have contacted the placers so they can verify.

We were very happy to find Strike It Rich, Homage to the Past, and
the Hopeful Miner. Some very fine carving on all 3. Also found the
Calling All Boxes hitchhiker, which we shall send on its way
elsewhere sometime soon.

There was a tourbus full of senior hikers in the preserve. It was
nice to see lots of others enjoying this magnificent location. And
there is no fee during the week.
